ANALYSIS

1. Pie Chart - Monthly Coffee Sales by Country: The pie chart displays the distribution of coffee sales by country. Each slice of the pie represents a country, and the size of the slice corresponds to the proportion of coffee sales for that country. In this chart, the data shows that Ecuador has the highest coffee sales, followed by Guinea and Bolivia. Papua New and Madagascar have comparatively lower sales. The chart is interactive, allowing users to filter the data using the range slider and toggle the 3D view of the chart with buttons.

2. Combo Chart - Monthly Coffee Production by Country: The combo chart displays monthly coffee production by country, combining both bar and line chart elements. The bars represent the monthly production of coffee for each country, while the line represents the average production. It provides a visual comparison of coffee production trends for different countries over multiple months. For example, in the year 2005/06, Ecuador had the highest production, while Rwanda had the lowest. The average production is represented by the line. The chart's title, axes labels, and chart types are appropriately labeled for clarity.

3. Confusion Matrix: The confusion matrix is a table that helps evaluate the performance of a classification model, typically used in machine learning. In this example, it shows the confusion between actual and predicted values, categorizing them as "Positive" and "Negative." It can be useful for assessing the accuracy of a predictive model by comparing how well it predicts positive and negative outcomes. For instance, in your data, there are 5 true positive predictions, 3 true negative predictions, 2 false positive predictions, and 1 false negative prediction.

4. Regression Table: The regression table provides information about the coefficients of a regression model. In this example, it displays coefficients for the intercept and the "Coffee Sales Monthly" variable. The coefficients represent the relationship between the predictor variable (Coffee Sales Monthly) and the target variable. In this case, the intercept is 0.75, indicating a base value, and the coefficient for "Coffee Sales Monthly" is 1.2, showing how changes in coffee sales affect the target variable. This table can be useful for understanding the linear relationship between variables in a regression analysis.
